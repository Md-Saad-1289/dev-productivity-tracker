# 💻 Dev Productivity Tracker

**Dev Productivity Tracker** হলো একটি All-in-One Productivity Web App যা আপনার দৈনন্দিন কাজ, Pomodoro sessions, ব্যয় এবং নোট ট্র্যাক করতে সাহায্য করে। এটি সহজ, responsive এবং modern UI সহ তৈরি করা হয়েছে।  

---

## 🌟 Features

- **Expense Tracker**  
  - Add, delete এবং filter expenses by category  
  - Pie chart visualization for expenses  
  - Clear all expenses option  

- **Pomodoro Timer**  
  - Start, pause, reset timer  
  - Set custom timer durations  
  - Track completed Pomodoros  
  - Visual progress bar  
  - Daily work hours tracking & chart  

- **Daily Goal Tracker**  
  - Set and track daily working hour goals  
  - Progress bar and summary  

- **Notes Section**  
  - Add, edit, delete notes  
  - Categorize notes  
  - Track date and time of each note  

- **Charts & Weekly Summary**  
  - Work hours bar chart  
  - Expense pie chart  
  - Weekly summary with total expenses, average hours, best day  

- **Export Options**  
  - Export weekly report as **PDF**  
  - Export weekly data as **CSV**  

---

## 📦 Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)  
- [Chart.js](https://www.chartjs.org/) — for charts  
- [jsPDF](https://github.com/parallax/jsPDF) — for PDF export  
- LocalStorage — for persistent data
